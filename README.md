# WWlab HPC Bioinformatics & AI4Bio Manual

本仓库用于维护王文实验室 HPC 生物信息与 AI4Bio 软件使用手册。

文档覆盖：

- HPC 使用规则、数据与存储管理、任务提交与日志管理；
- 基础生信工具、基因组学、比较基因组学、多组学、单细胞与空间组学；
- 蛋白结构预测、蛋白设计、分子设计、AI4Bio 大模型与多模态工具；
- 数据库、模型权重、软件条目模板和常见错误排查。

## 本地预览

```bash
pip install -r requirements-docs.txt
mkdocs serve
```

## 构建文档

```bash
mkdocs build
```
